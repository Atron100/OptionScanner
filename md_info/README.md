# OptionScanner

Personal professional options analysis platform.

See **PROJECT_SPEC.md** for the full specification.

## Initial Features

- Interactive Brokers integration
- Option chain downloader
- Greeks and IV analytics
- Strategy optimizer
- Income strategy scanner
- Payoff graphs
- Portfolio analytics

## Initial Strategies

- Iron Condor
- Iron Butterfly
- Covered Call (CC)
- Cash Secured Put (CSP)
- PMCC
- Calendar
- Double Calendar
- Diagonal
- Double Diagonal
- Vertical Credit/Debit Spreads
- Butterfly
- Broken Wing Butterfly
- Ratio Backspread

## Long-term Goal

Build a production-quality options platform capable of screening, ranking, optimizing, and eventually executing options strategies using IBKR.
